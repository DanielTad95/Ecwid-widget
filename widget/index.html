Ecwid Widget Files v2.0

Widget Server Active - Modular Architecture

This server hosts the JavaScript and CSS files for the Ecwid Recently Updated Products widget with improved modular architecture.

Available Files:

  • widget.js
    Main widget loader (NEW v2.0) - Loads all modules automatically
  • recently-updated-legacy.js
    Legacy monolithic widget file (fallback)
    • Main widget JavaScript file that integrates with Ecwid stores
  • styles.css
    CSS styles for the widget that match Ecwid's design standards

    • Widget Modules (v2.0):

    • config.js
      Configuration and constants
    • logger.js
      Environment-aware logging system
    • api.js
      API communication module
    • dom.js
      DOM manipulation and caching
    • ui.js
      HTML generation and sanitization
    • tracking.js
      Analytics and order tracking
    • core.js
      Main coordination logic

    Integration Instructions:

    For v2.0 Modular Widget:

    1. Host these files on a public HTTPS server
    2. Include the main widget loader:
      3. <script src="http://localhost:3002/widget.js"></script>
    3. Widget will automatically load all modules and initialize

    For Legacy Fallback:

    <script src="http://localhost:3002/recently-updated-legacy.js"></script>
